# Collatz Conjecture Calculator

コラッツ予想を計算するシンプルなウェブアプリケーションです。

## 概要

コラッツ予想（3n+1問題）は、任意の正の整数に対して以下の操作を繰り返すと、必ず1に到達するという予想です：

- 偶数なら2で割る
- 奇数なら3倍して1を足す

このアプリケーションでは、入力した数値のコラッツ予想の計算過程を視覚的に表示します。

## 開発環境のセットアップ

1. 依存関係のインストール:
```bash
bun i
```

2. ビルドの実行:
```bash
bun run build
```
